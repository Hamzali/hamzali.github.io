Bilgisayarınızda bir dizini takip eder ve internet üzerindeki deponuzla bağlar.

Repository??

İnternet üzerindeki projenin saklandığı yer. Ana depo.

Stage??

Yaptığınız değişiklerin depoya yollanacaklarıdır. Sadece kendi bilgisayarınızda kalması gerekenleri stage yapmazsınız.

Commit??

Kendi bilgisayarınızda olan bir projeyi takibe alma ve repo kurulumu

  1. Önce proje dizininizde komut satırı açın.
  2. Projenin takibinin başlaması için şu komutu girin:
    git init
  3. Sonra bu değişikliklerden commitlemek hepsini stagelemek için:
    git add -A
  4. Değişiklikler paketlenip göndermeye hazır yani commitlemeye:
    git commit -m "**anlamlı mesaj"
  5. Şimdi bu değişikleri internetteki depoya yollama zamanı. Bir dakka internetteki depo nerde??
  6. Hemen github'a üye ol ve bir tane repository aç ama initialize etme.
  7. Sonra kara kuru komut satırına dön.
  8. Şimdi internetteki ana depo adresini bilgisayardaki git programına söyle:
    git remote add origin https://github.com/user/repo.git
  9. Evet artık tüm değişiklikleri ana merkeze yollayabiliriz:
    git push -u origin master
    **sadece ilk seferde bu şekilde sonra sadece git push olarak kullanabilirsiniz.

Ve artık zaten Bilgisayarınızda olan bir projeyi git ile takip edip internete taşıdınız.

Reposu olan bir projeyi bilgisayarınıza alma

  1. Reponun linkini bulduktan sonra içine kurmak istediğiniz dizine gidip:
    git clone https://github.com/user/repo.git
  2. Bu kadar! artık her değişiklik yaptığınızda sırayla;
    stage(add) - commit - push yapın.
  3. Başkalarının değişimlerini almak için ise:
    git pull
